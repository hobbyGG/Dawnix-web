## 🚀 使用方式

### 前端开发

1. 进入流程定义 → 新建模型
2. 切换到 "流程编排" 标签
3. 添加 "用户任务" 节点
4. 在右侧属性面板选择审批人
5. 支持多选、搜索、全选、清空

### 后端适配

需要确保以下功能：

1. `/enum/approvers` 接口返回活跃用户列表
2. 流程定义保存时支持 `candidates.users` 数组
3. 定义多个审批人的审批流转规则

### 数据格式

**保存流程时发送的格式**:
```json
{
  "nodes": [{
    "type": "user_task",
    "candidates": {
      "users": ["uid_1", "uid_2", "uid_3"]
    }
  }]
}
```
